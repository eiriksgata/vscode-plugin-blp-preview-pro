# 地形编辑工具使用指南

## ✨ 功能概览

已为 `.w3e` 地形文件添加**实时编辑**能力（之前仅支持预览）。

### 新增功能
- ✅ **高度编辑** - 4 种笔刷模式（提高、降低、平滑、平铺）
- ✅ **实时渲染** - GPU 缓冲增量更新，零延迟反馈
- ✅ **撤销/重做** - 完整的撤销栈支持
- ✅ **文件保存** - 直接保存回 W3E 文件
- ✅ **工具栏 UI** - 笔刷大小/强度可视化控制

### 相机控制（同预览模式）
| 操作 | 效果 |
|------|------|

| 操作 | 效果 |
|------|------|

| **Ctrl+Shift+Z** / **Cmd+Shift+Z** | 重做 |
| **Ctrl+S** / **Cmd+S** | 保存 |

## 🖌️ 笔刷模式详解

### 1️⃣ 提高（Raise）— 按键 `1`
- **效果**：抬高地形
- **典型用途**：创建山脉、丘陵、防御墙
- **参数影响**：强度越大，抬升越明显

### 2️⃣ 降低（Lower）— 按键 `2`
- **效果**：降低地形
- **典型用途**：挖沟、创建河流、凹陷地形
- **强度控制**：控制下压幅度

### 3️⃣ 平滑（Smooth）— 按键 `3`
- **效果**：平均周围点，平滑过渡
- **典型用途**：消除陡坡、平滑边界
- **强度含义**：混合比例（强度 1.0 = 完全平均，0.1 = 微调）

### 4️⃣ 平铺（Flatten）— 按键 `4`
- **效果**：将笔刷范围内的点平铺到高度 0
- **典型用途**：快速制作平坦区域（营地、竞技场）
- **强度**：无效（直接置为 0）

## 🎛️ 工具栏参数

### 笔刷大小（1-10）
- **值域**：1-10 个角点半径
- **默认**：3
- **效果**：控制一次笔刷作用的范围
  - 1 = 只影响中心点
  - 5 = 直径 10 角点
  - 10 = 直径 20 角点（大区域）

### 笔刷强度（0.1-2.0）
- **值域**：0.1-2.0
- **默认**：0.5
- **效果**：单次操作的改变量
  - 0.1 = 柔和，需多次笔刷
  - 1.0 = 标准强度
  - 2.0 = 激进，一笔成型

### 距离衰减
笔刷使用高斯衰减（距离中心越远，效果越弱）：
```
效果 = 笔刷强度 × max(0, 1 - 距离/半径)
```

## 💾 制工作流程

### 典型编辑流程

```
1. 打开 map.w3x（或分解后的地图文件夹）中的 war3map.w3e
↓
2. 使用笔刷编辑地形
   - 选择合适的笔刷模式（按 1-4）
   - 调整大小和强度
   - 左键拖动绘制
↓
3. 不满意？撤销（Ctrl+Z）
↓
4. 完成修改后，按 Ctrl+S 保存
↓
5. 重新打包或导出地图
```

### 例子：创建山峰

```
1. 预选笔刷模式：提高（按 1）
2. 调整大小：7-8（中等笔刷）
3. 调整强度：1.0-1.5（较强）
4. 在目标位置左键按住，上下左右拖动，多次笔刷叠加
5. 用平滑模式（按 3）消除陡坡
6. Ctrl+S 保存
```

## ⚙️ 技术架构

### 核心模块

| 文件 | 职责 |
|------|------|
| `picker.ts` | 射线拾取系统（Möller–Trumbore 算法）|
| `terrain-editor.ts` | 编辑逻辑、撤销栈、笔刷算法 |
| `index.html` | 工具栏 UI、快捷键绑定 |
| `index.ts` | 前端入口、事件循环 |

### 渲染优化
- **GPU 缓冲增量更新**：仅修改变化的高度图区域，而非全量重传
- **距离衰减**：使用高斯函数，从中心向外衰减
- **脏标记**：仅在有修改时更新 GPU 数据

### 撤销实现
- **撤销栈**：记录每次笔刷的所有 `HeightChange[]`
- **空间优化**：只存储改变的角点，而非整张地图快照
- **操作原子性**：单次拖动 = 一个操作（可一次性撤销）

## ⚠️ 已知限制 & 未来计划

### 当前限制
- ❌ 不自动重计算悬崖（修改高度后悬崖显示可能不同步）
  - 解决方案：保存后重新打开文件，或手动调用 `isCliff()` 逻辑
- ❌ 贴图编辑未实现（目前只能编辑高度）
- ❌ 多地图同时编辑无冲突检测

### 未来计划
- 🔲 **贴图编辑**：直接更改地面/悬崖贴图 ID
- 🔲 **水面编辑**：修改水位、水深颜色
- 🔲 **悬崖自动重计算**：修改高度后自动更新悬崖模型
- 🔲 **高度图导入/导出**：支持 HGT、RAW 等格式
- 🔲 **高度图预览**：小地图视图显示高度分布

## 🐛 故障排查

### 笔刷没有反应
- 确认左键处于按下状态
- 检查是否在 3D 视图中操作
- 查看浏览器控制台是否有 JavaScript 错误

### 修改没有保存
- 确认点击了"保存"或按 Ctrl+S
- 检查文件权限（是否有写入权限）
- 确认文件路径正确（`.w3e` 扩展名）

### GPU 缓冲更新卡顿
- 减小笔刷大小（1-3 最快）
- 降低笔刷强度
- 关闭其他占用 GPU 的应用
