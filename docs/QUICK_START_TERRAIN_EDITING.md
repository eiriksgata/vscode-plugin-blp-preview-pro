# 🎮 地形编辑快速入门

## ⚡ 5 秒快速开始

1. **打开地图** → 在 VS Code 中打开 `.w3e` 文件
2. **选择笔刷** → 点击工具栏的 🔼 🔽 〰️ ▭ 按钮（或按 1-4 键）
3. **左键拖动** → 在 3D 画面中左键按住不放，拖动鼠标
4. **Ctrl+S 保存** → 修改完成后保存

## 🎨 4 种笔刷模式详解

### 1️⃣ 🔼 提高（raise）→ 按键 1
**作用**：向上抬起地形
**使用场景**：
- 制作山脉、丘陵
- 创建防御墙、城墙
- 抬高整个地区

**操作**：在目标位置左键拖动，持续施加笔刷，高度不断增加

---

### 2️⃣ 🔽 降低（lower）→ 按键 2
**作用**：向下压低地形
**使用场景**：
- 挖掘河流、沟渠
- 创建凹陷地形（天坑、陨石坑）
- 制作水渠

**操作**：左键拖动，地形下降

---

### 3️⃣ 〰️ 平滑（smooth）→ 按键 3
**作用**：平均周围高度，平滑陡坡
**使用场景**：
- 消除锯齿状边界
- 制作平缓坡道
- 美化陡峭山坡

**提示**：强度越高，平滑效果越明显；弱强度可微调过渡

---

### 4️⃣ ▭ 平铺（flatten）→ 按键 4
**作用**：将笔刷范围内的点都设为高度 0
**使用场景**：
- 快速制作平坦竞技场
- 清除不规则地形
- 制作建筑基地

**特性**：强度无效（直接平铺）

---

## 🔧 工具栏参数调整

### 笔刷大小 1-10
```
小（1-3）    → 精细编辑，修改小范围
中（4-6）    → 平衡，适合大多数场景
大（7-10）   → 快速整体修改，覆盖大区域
```

**推荐**：
- 新手：3-4（易控制）
- 造山脉：6-8（笔画粗放）
- 细节调整：1-2（精确）

### 笔刷强度 0.1-2.0
```
弱（0.1-0.5）   → 柔软，需要多次笔刷叠加
强（1.0）        → 标准强度，推荐
激进（1.5-2.0） → 一笔成型，易出现陡坡
```

**推荐**：
- 平滑模式：0.2-0.5（微调）
- 提高/降低：0.8-1.2（平衡）
- 快速整平：1.5+ （效率优先）

## 📋 完整工作流示例

### 场景：制作一个山峰

```
第 1 步：选择 🔼 提高模式（按 1）
第 2 步：调整大小为 6，强度为 1.0
第 3 步：在目标位置左键单点，松开
      → 中心点上升，周围渐弱
第 4 步：重复 3-4 次单点，堆积高度
      → 形成山峰
第 5 步：顶部陡峭？
      → 切换到 〰️ 平滑模式（按 3）
      → 在山顶拖动几下，平滑边缘
第 6 步：满意？
      → Ctrl+S 保存
```

### 场景：制作河流

```
第 1 步：选择 🔽 降低模式（按 2）
第 2 步：调整大小为 4-5，强度为 0.8
第 3 步：沿着预定路线拖动鼠标
      → 形成一条沟渠
第 4 步：如果太宽？
      → 减小笔刷大小到 2-3，再精修
第 5 步：两边陡？
      → 切换 〰️ 平滑，轻轻拖过两侧
第 6 步：Ctrl+S 保存
```

## 🎓 初学者常见问题

### Q: 怎么知道我在编辑哪个位置？
**A**：看右侧信息面板！它会实时显示：
- 鼠标下的 **角点位置** (column, row)
- 该点的 **当前高度**
- 当前 **笔刷参数**

### Q: 拖动时不小心拉太高了？
**A**：按 `Ctrl+Z` 撤销。可以撤销无限多步！

### Q: 怎么快速抹平一大块地？
**A**：
1. 按 `4` 切换到 ▭ 平铺模式
2. 增大笔刷大小（7-10）
3. 在目标区域点一下

### Q: 一次笔刷不够强？
**A**：增大 **笔刷强度**，或多拖几次。强度 2.0 + 大小 8 = 最强效果

### Q: 保存后重新打开发现不一样？
**A**：悬崖模型可能需要重新计算。这是已知限制，暂时的解决方案是：
- 保存后关闭再打开文件
- 或接受轻微的显示偏差（实际高度数据已正确保存）

## 💡 进阶技巧

### 1. 多次轻拍创建平缓过渡
```
不推荐：一次强笔刷，陡坡
推荐：多次弱笔刷，缓坡
    弱强度（0.3-0.5）× 5-10 次
```

### 2. 先高后平的工作流
```
第 1 步：用强度 1.5+ 快速抬高/压低到大概位置
第 2 步：用强度 0.3-0.5 的平滑模式微调细节
```

### 3. 使用不同大小层叠创建复杂地貌
```
大笔刷（8-10）→ 粗略整体形状
中笔刷（4-6）  → 中等细节
小笔刷（1-3）  → 精细纹理
```
